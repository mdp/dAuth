## Installation

Still very alpha

1. `git clone https://github.com/mdp/dAuth.git`
1. `cd dAuth`
1. `npm install`
1. `npm intall -g react-native-cli`

#### iOS

Install via XCode

#### Android

1. `react-native run-android`
